# Handover Agent 

An **LLM-based staff handover agent** prototype with three main functions:

1. **Summary Report** (`/summary-report`)  
   Generates short handover summaries from ingested documents.

2. **Chatbot (RAG-based)** (`/chatbot`)  
   Answers questions as if it were the predecessor, grounded in retrieved evidence.

3. **Scheduling & Task Extraction** (`/scheduling`)  
   Extracts tasks, owners, and due dates from free text or attachments.
